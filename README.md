 Hospital Emergency Room Interactive Dashboard
📌 Project Overview

This project focuses on building an interactive Hospital Emergency Room (ER) Dashboard to analyze patient flow, waiting times, admissions, and department referrals.
The goal is to transform raw healthcare data into actionable insights that support operational decision-making and improve patient experience.

This dashboard was created as part of my end-to-end data analytics learning, where I worked across the complete analytics lifecycle — from data cleaning to insight delivery.

🎯 Objectives

Analyze ER performance using key healthcare metrics

Identify patient wait time and delay patterns

Understand admission trends and patient demographics

Visualize department-wise referrals

Design a user-friendly, interactive dashboard for stakeholders

🔄 End-to-End Workflow

Business Understanding

Defined healthcare KPIs relevant to ER operations

Identified decision-maker requirements

Data Cleaning & Preparation

Handled missing and inconsistent values

Standardized date, age group, and categorical fields

Data Analysis

Admission vs non-admission trends

Age group-wise patient distribution

Delay vs on-time attendance analysis

Dashboard Design

Interactive slicers (Year, Month)

KPI cards and comparative visuals

Clean layout for quick insight consumption

Insight Communication

Highlighted operational bottlenecks

Enabled data-driven decision support

📊 Key Metrics & Insights

Total number of ER patients

Average patient waiting time (minutes)

Patient satisfaction score

Admission vs not admitted ratio

Patients attended within time vs delayed

Age group-wise patient distribution

Department-wise referral analysis
